# گرفتن APK آنلاین ماه‌چین

این نسخه دو خروجی می‌سازد:

- `MahChin-debug-apk`: فقط برای تست توسعه
- `MahChin-release-apk`: نسخه Release امضاشده آزمایشی، مناسب‌تر برای نصب روی گوشی

## مراحل

1. فایل‌های پروژه را در GitHub آپلود کن.
2. وارد تب **Actions** شو.
3. روی **Build MahChin APK** بزن.
4. دکمه **Run workflow** را بزن.
5. صبر کن سبز شود.
6. پایین صفحه، از بخش **Artifacts** فایل `MahChin-release-apk` را دانلود کن.
7. فایل ZIP دانلود شده را Extract کن.
8. فایل `app-release.apk` را روی گوشی نصب کن.

## نکته مهم نصب

اگر نسخه قبلی ماه‌چین روی گوشی نصب است، اول حذفش کن. چون نسخه جدید Release با امضای متفاوت ساخته می‌شود و ممکن است روی نسخه Debug نصب نشود.

اگر Play Protect هشدار داد، دلیلش این است که برنامه از گوگل‌پلی نصب نشده. برای تست شخصی می‌توانی گزینه نصب با مسئولیت خودت را انتخاب کنی. برای انتشار رسمی باید نسخه Release نهایی با کلید اختصاصی خودت ساخته شود.
